Difference between div and span?

Div is a block element which always starts on a new line and takes up the full width of the browser.

Span is an inline element which does not starts on a new line and only takes up as much width as necessary.

Question no:-5

Difference between block and inline?

Block level element always starts in a new line and always takes up the full width of the browsers and browser automatically add some space at left and right side of the element which is call as margin.

Example:-
<div> is a block level element.

<p> is a block level element.

Inline Element does not start in a new line and only takes as much width as necessary.

Example:- <span> is a inline element.  <img> is inline element

Question no:-6

Difference between em and strong?

The <strong> tag is used to define text with strong importance, and the content inside the strong tag displayed in bold. whereas <em> tag is used to define the emphasized text. the content inside is typically displayed in italic.Both tags are used atleast two or three times in a paragraph to show the importance of the text that is written in paragraph.

Difference between HTML tags and elements?

What are semantic tags?
